# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Loomcraft, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities
2. Email **dev@voidcorp.io** with a description of the issue
3. Include steps to reproduce if possible

We aim to acknowledge reports within 48 hours and provide a fix within 7 days for critical issues.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.x     | Yes       |
| < 2.0   | No        |

## Scope

This policy covers the `@loomcraft/cli` npm package and the Loomcraft web application.
